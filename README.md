### Setup

- Clone a aplicação em seguida instale as dependencias do projeto:
<br>
    sudo npm install
<br>
    sudo npm run dev
<br>
- Crie um Alias no .Bash para rodar Sail Artisan:
<br>
    alias sail='bash vendor/bin/sail'
<br>
- Crie e Execute os containers do Docker:
    sail up -d
